一、项目介绍
---
系统包含两种角色：用户、管理员，系统分为前台和后台两大模块，主要功能如下：

### 1 管理员模块的实现
- 美容项目管理: 管理美容项目的添加、修改、删除及查询操作。
- 产品库存管理: 查看、修改、删除及查询产品库存信息。
- 产品购买管理: 审核产品购买信息。
- 会员卡管理: 管理会员卡的信息，包括添加、修改、删除及查询操作。

### 2用户模块的实现
- 项目预定管理: 用户可以进行项目预定操作。
- 产品购买信息: 用户可以进行产品购买及支付操作。
- 会员充值管理: 用户可以进行会员充值操作。
